Как я могу заимповать и распаковать массивы в JavaScript?

Содержание

  1. Как я могу заимповать и распаковать массивы в JavaScript?
  2. Zip-массивы
  3. Распаковка массивов
  4. Замкнойте массив в объект

Схимпирование двух или более массивов относится к объединению их элементов в один массив массивов. Поскольку это звучит довольно расплывчато, давайте посмотрим на пример:

JavaScript

const arrays = ['a', 'b'], [1, 2], [true, false];

const zipped = [['a', 1, true], ['b', 2, false]];

JavaScript

const arrays = ['a', 'b'], [1, 2], [true, false];

Как видите, этот первый массив в этом примере содержит все первые элементы исходных массивов, второй массив содержит все вторые элементы и так далее.
